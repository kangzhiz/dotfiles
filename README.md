# dotfiles

Just some of my dotfiles and scripts.

Currently using [Fedora 33 Workstation w/ GNOME](https://getfedora.org/).

### Setup

1. Clone repository into `~/Documents/dotfiles`.
2. Navigate to `~/Documents/dotfiles`.
3. Run `install.sh`.
